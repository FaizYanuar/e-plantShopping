# Paradise Nursery

**Paradise Nursery** is a front-end e-commerce web application designed for houseplant enthusiasts. This project provides a seamless shopping interface where users can explore a diverse catalog of plants, manage their selections, and prepare for checkout. 

Built as a comprehensive final project, this application demonstrates dynamic state management and interactive component design across multiple views.

## Key Features

* **Landing Page:** A welcoming entry point that introduces the Paradise Nursery brand and invites users to shop.
* **Product Listing Page:** A dynamic browsing interface where users can view a variety of available houseplants and easily add them to their shopping cart.
* **Interactive Shopping Cart:** A dedicated cart management page that displays all selected items, calculates the total cost in real-time, and allows users to dynamically adjust item quantities prior to checkout.

## Technologies Used

* **React.js:** Used to build the interactive, component-based user interface.
* **Redux Toolkit:** Used for global state management (handling the shopping cart, item quantities, and total costs).
* **Vite:** A fast build tool and development server.
* **CSS:** Custom styling for a responsive and visually appealing layout.
